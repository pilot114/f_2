Gateway - набор интерфейсов для гибкого внедрения
низкоуровневого (инфраструктурного) кода в высокоуровневый (бизнесовый) код.  
Некоторые гейтвеи подключаются из внешних зависимостей, поэтому тут они явно не объявлены.  
Вот их список:

| Для чего       | Стандартное название | Откуда подключается (реальное название)  |
|----------------|----------------------|------------------------------------------|
| Создание писем | WriteEmailGateway    | Symfony\Component\Mailer\MailerInterface |
| Запись в БД    | WriteDatabaseGateway | Database\Connect\WriteDatabaseInterface  |
| Чтение из БД   | ReadDatabaseGateway  | Database\Connect\ReadDatabaseInterface   |
| Кэширование    | CacheGateway         | Psr\Cache\CacheItemInterface             |

> Гейтвеи не обязательно делятся на чтение и запись, если состояние, с которым они работают,
является служебным. Например, логи и кэш могут быть удалены в любой момент без
нарушения работоспособности кода (более того, не важно где именно они хранятся - в специальном
хранилище или в RAM) - для них принцип разделения интерфейсов менее значим.
