# Создание нового бэкенда корпоративного портала

## Дата создания
24.01.2024

## Статус
Принятый

---

## Контекст

Корпоративный портал имеет устаревший бэкенд, с большим количеством legacy кода, что затрудняет его поддержку и развитие.
Необходимо создать новый бэкенд, который будет более современным, масштабируемым и удобным для поддержки.

## Решение

- **Взять фреймворк Symfony в качестве отправной точки, как наиболее подходящий для корпоративного проекта**
- **Спроектировать простое и понятное API на основе json-rpc 2.0**
- **Использовать актуальную версии PHP (8.2)**
- **В течении периода внедрения - созвоны каждые две недели для актуализации статуса разработки**

---

## Альтернативные варианты

- Эволюционная разработка текущего бэкенда - уже было несколько попыток, но каждый раз мы упирались в ограничения
legacy кода, в результате чего сдвигались сроки, а качество оставляло желать лучшего.

## Последствия

- Подготовлен план разработки и внедрения нового бэкенда (февраль 2024 - июль 2024)
- Август 2024 - запуск в продакшен, все новые фичи разрабатываются на новом бэкенде
